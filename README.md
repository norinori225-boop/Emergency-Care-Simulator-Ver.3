# 救急隊 観察シミュレーター Ver.3

救急隊員向けの傷病者観察・評価・処置訓練ツールです。インストール不要、ブラウザだけで動作します。

## 概要

実時間でバイタルサインが変動する傷病者に対して、状況評価・初期評価・問診・全身観察・処置・搬送判断の一連の流れを訓練できるシミュレーターです。処置の手順・抜け・タイミングが採点され、終了後には鑑別診断と病態変化の解説が表示されます。

## デモ

ファイルをダウンロードしてブラウザで開くだけで動作します。サーバー不要・インターネット接続不要。

## 機能

- **リアルタイムバイタル変動** — HR / 血圧 / SpO₂ / 呼吸数 / 意識 が時間経過と処置内容に連動して変化
- **ECG・SpO₂波形表示** — 心拍数に応じたキャンバス描画
- **アラーム** — 異常値で警告点滅
- **7タブの観察フロー**
  - 状況評価 / 初期評価 / 問診(SAMPLE/OPQRST) / 家族・関係者 / 全身観察 / 処置 / 判断・搬送
- **症例ガチャ** — カテゴリ・レベル別にランダム症例を生成（年齢・性別・初期バイタルが毎回変化）
- **活動記録タイムライン** — 全操作・処置・イベントを時刻付きで記録
- **採点・デブリーフィング** — 終了後にスコアとフィードバックを表示
- **鑑別診断と除外根拠** — 終了後に主要な鑑別疾患と「なぜその疾患が考えにくいか」を提示
- **病態変化の解説** — 「なぜ悪化したか」「なぜ処置で改善したか」を終了後に解説
- **根拠の選択（rationale）** — 疾患推定の根拠となる所見を選ばせ、観察力を評価
- **一時停止** — 教育場面での解説停止に対応

## 使い方

1. `index_ver3.html` をブラウザで開く
2. 固定症例カードを選ぶか、症例ガチャでカテゴリ・レベルを指定して出題
3. 「状況評価」→「初期評価」でバイタルをモニターに表示
4. 問診・全身観察で情報収集
5. 必要な処置を実施
6. 疾患推定・搬送先・根拠所見を選び「観察終了・評価へ」
7. スコア・鑑別診断・病態解説でデブリーフィング
