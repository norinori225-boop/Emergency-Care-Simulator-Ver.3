# 救急隊 観察シミュレーター

救急隊員向けの傷病者観察・評価・処置訓練ツールです。インストール不要、ブラウザだけで動作します。

## 概要

実時間でバイタルサインが変動する傷病者に対して、状況評価・初期評価・問診・全身観察・処置・搬送判断の一連の流れを訓練できるシミュレーターです。処置の手順・抜け・タイミングが採点されます。

## デモ

ファイルをダウンロードしてブラウザで開くだけで動作します。サーバー不要・インターネット接続不要。

## 収録シナリオ

| ID | 主訴 | 推定疾患 | 難易度 |
|----|------|----------|--------|
| ami | 胸が苦しい60代男性 | 急性心筋梗塞(ACS) | 中等症〜重症 |
| cva | ろれつが回らない70代女性 | 脳梗塞 | 重症 |
| ana | 蜂に刺された後の呼吸困難 | アナフィラキシーショック | 重症・分単位で悪化 |

## 機能

- **リアルタイムバイタル変動** — HR / 血圧 / SpO₂ / 呼吸数 / 意識 が時間経過と処置内容に連動して変化
- **ECG・SpO₂波形表示** — 心拍数に応じたキャンバス描画
- **アラーム** — 異常値で警告点滅
- **7タブの観察フロー**
  - 状況評価 / 初期評価(ABCD) / 問診(SAMPLE・OPQRST) / 家族・関係者 / 全身観察 / 処置 / 判断・搬送
- **活動記録タイムライン** — 全操作・処置・イベントを時刻付きで記録
- **採点・デブリーフィング** — 終了後にスコアとフィードバックを表示
- **一時停止** — 教育場面での解説停止に対応

## 使い方

1. `index.html` または `救急隊観察シミュレーター_統合版.html` をブラウザで開く
2. 症例を選んで「開始」
3. 「状況評価」→「初期評価」でバイタルをモニターに表示
4. 問診・全身観察で情報収集
5. 必要な処置を実施
6. 疾患推定・搬送先を選び「観察終了・評価へ」

- モダンブラウザ（Chrome / Edge / Firefox / Safari）
